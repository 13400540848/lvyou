1.限号风险提示：

在当期销售过程中，当参与号码未达到当前允许销售的最大参与数额时，各限号参与方式按实际销售的参与数额实时从福彩中心数据库限号池获取参与许可；若参与号码达到当前允许销售的最大参与数额，则限号池停止向各限号参与方式分配该参与号码的许可。

按各参与方法不同设定最大赔付量。该参与方法最大参与注数=（该参与方法销量*0.59+该参与方法最大赔付量）/单注奖金，当该参与方法达到最大参与注数量时，出现限号。不同参与方式风险控制量不同，单种参与方式的最大赔付量是50万DG，单期最大返奖额可超过300万DG！

动态实时限制是指在销售过程中，随着销量的增长，每种参与方式的即时允许销售注数，动态实时限制随着该参与方式销售总额的变化，按规定的算法进行动态计算并加以限制的过程。

在当期销售过程中，当所选参与号码参与注数未超过当前动态实时限制注数时，该参与视为有效参与；若所选参与号码参与注数已超过当前动态实时限制注数，则视为无效参与，此时销售终端不接受该参与的申请。

2.追号异常处理流程：

由于官方开奖延迟或网络故障导致的用户追号方案未能按照用户要求中奖停止，由此产生的未停止期次参与损失，将由网易彩票于异常发生的第二个工作日，向用户补偿追号参与损失差额。
